# Flapjack Skills

Agent skills for integrating [Flapjack](https://flapjack.chat) AI agents into your applications.

## Available Skills

### flapjack-integration

Everything you need to embed Flapjack agents — SDK setup, React hooks, streaming, custom tools, and the full API reference.

**Install:**

```bash
npx skills add Maatskappy/flapjack-skills@flapjack-integration
```

## What are Skills?

Skills are modular packages that give AI coding agents (Claude Code, Codex, Gemini CLI) specialized knowledge about specific tools and frameworks. Once installed, your agent knows how to use the Flapjack SDK correctly — types, patterns, common mistakes, and all.

Browse more skills at [skills.sh](https://skills.sh).
